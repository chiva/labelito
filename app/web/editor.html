Template format reference

A quick cheatsheet. The full reference (every parameter, defaults, and bounds) lives in docs/template-format.md in the repo.

Top-level keys

name (unique id), description, label (a label id from the reference above), and layout are required. rotate (0/90/180/270) and fields are optional.

Fields & tokens

Declare fields.required/fields.optional (names: letters, digits, _). Reference each with {{name}} in an element's text, data, or name. Computed tokens are always available: {{date}}, {{now}} (with ±Nd/w/m/y offset and :strftime format), and {{seq}} (sequence batches). Translation words use [[key]].

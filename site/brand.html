01 / brand story

Homelab-grade tooling that respects your time.

labelito is for the tinkerer who runs their own stack. It does one thing — print labels on a Brother QL — and does it cleanly: declarative templates, a live preview, a tidy HTTP API. No cloud account, no app store, no lock-in. The brand should feel the same way it works: precise, unfussy, and quietly friendly.

// one line
The elevator pitch

"Self-hosted, declarative label printing for Brother QL printers — a YAML template and one HTTP call away from a printed label."

// who it's for
The audience

Homelabbers, Home Assistant users, makers, and small teams automating physical labels — freezer bags, cables, asset tags, shelves.

// the feeling
The vibe

A well-labeled workshop. Everything in its place, nothing precious. Confident but never corporate.

Declarative, not clicky

A label is data. Define it once, print it forever.

Fail closed, speak plainly

Safe defaults, honest errors, no phantom success.

Yours to run

On your hardware, your network, your terms.

Small on purpose

Does one thing well; refuses to sprawl.

05 / iconography

Icons that survive a label

labelito prints at 300 dpi in pure black-on-white, so its icons are bold, solid, single-weight glyphs — legible at 12 mm. They come from three bundled collections; no bespoke drawing. In the UI they sit on blue; on a label they render 1-bit.

06 / voice & tone

How labelito talks

Like a helpful maintainer in a good README: direct, concrete, a little dry. It assumes you're capable, skips the marketing, and always tells you what actually happened.

Plain

Short words, real nouns. "Print", not "initiate a print job".

Precise

Name the media, the endpoint, the code. Numbers over adjectives.

Honest

Say when something failed and why. Never fake success.

Warm

A dry aside is fine. Condescension never is.

✓ say this
"Printed on QL-810W in 0.4s."
"Loaded roll is 62mm continuous — this template needs 62×29 die-cut."
"Drop a .yaml in templates/ and hot-reload — no restart."
✗ not this
"Your print job has been submitted successfully!"
"Oops! Something went wrong. Please try again later."
"Unlock powerful, seamless labeling solutions today."

08 / code & cli

Code & CLI style

Code is a first-class brand surface. Terminals and snippets always use JetBrains Mono, a green $ prompt, blue strings, and a green → ✓ for results. Keep examples runnable and short.

terminal
$ curl -X POST localhost:8765/print \
    -d '{"template":"freezer-icon",
         "fields":{"title":"Bolognese"}}'

→ rendered + printed on QL-810W ✓ 0.4s
templates/62-freezer-icon.yaml
name: freezer-icon
label: "62"
layout:
  - {type: icon,  name: snowflake}
  - {type: title, text: "{{title}}"}
  - {type: text,  text: "[[frozen]]: {{date}}"}

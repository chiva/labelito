// how it works

Define once. Print forever.

1 · templates/62-freezer-icon.yaml
name: freezer-icon
label: "62"
layout:
  - {type: icon, name: snowflake}
  - {type: title, text: "{{title}}"}
  - {type: line}
  - {type: text, text: "Frozen: {{date}}"}
2 · one HTTP call
POST /print
{
  "template": "freezer-icon",
  "fields": {
    "title": "Bolognese"
  }
}
{{date}} resolves at render time — callers never pass dates.
3 · a printed label
Bolognese sauce
2 portions
Frozen: 07/01/2026
✓ printed · 0.4s

Computed dates
